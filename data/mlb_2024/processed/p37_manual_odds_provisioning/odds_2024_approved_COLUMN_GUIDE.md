## Deriving p_market from closing odds
For a moneyline market with no-vig adjustment:

```
raw_p = 1 / odds_decimal_home
# Apply standard two-sided vig removal if both sides available:
raw_q = 1 / odds_decimal_away
overround = raw_p + raw_q
p_market = raw_p / overround
```

Never infer `p_market` from the game outcome.
Never set `p_market = 1.0` for wins or `0.0` for losses.
