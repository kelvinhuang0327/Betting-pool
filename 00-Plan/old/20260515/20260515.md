今天建議不要直接跑完整 24H 長任務，先跑短版 **P1.5A：License Gate Only**。

優先順序如下：

1. **先做 MANUAL_REVIEW_REQUIRED 候選來源的條款確認**
   目的：避免花時間下載資料後才發現不能研究使用或不能留存。

2. **判斷是否存在 `ACCEPTABLE_FOR_LOCAL_RESEARCH` 或 `ACCEPTABLE_FOR_FIXTURE_ONLY` 來源**
   若沒有，直接維持 `RESEARCH_ODDS_DOWNLOAD_DEFERRED`，不要跑 join。

3. **若有合法來源，再做 5–20 筆 local-only join smoke**
   只放 local-only，不 commit raw CSV。

4. **若沒有合法來源，轉向 manual import / paid licensed dataset 決策**
   不要繼續在不明授權 public data 上消耗工程時間。

5. **暫時不要 push，除非使用者明確 YES**
   目前本地 commit 已完成，但 push 需授權。
