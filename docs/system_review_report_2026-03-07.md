## 一、執行摘要 (Executive Summary)

本系統已具備**生產級混合集成架構**，包含 10+ ML 模型、50,000 次蒙特卡洛模擬、7 階段機構決策閘與 Kelly 準則風控。工程成熟度高，但核心風險集中在以下三點：

| 風險 | 描述 | 嚴重程度 |
|---|---|---|
| 回測數據不足 | 僅 2023 WBC ~20 場，遠低於統計顯著門檻 50 場 | 🔴 極高 |
| 合成數據污染 | 真實數據不足時自動回退合成數據，嚴重影響回測可信度 | 🔴 極高 |
| 固定集成權重 | Elo 永遠 45%，不隨模型近期表現動態調整 | 🔴 高 |

## 二、現有系統架構評估

### 2.1 架構強項 ✅

- **資料隔離**: 嚴格無前視偏差，預測函數僅接收開賽前特徵
- **多模型共識**: 7 個獨立子模型 + Stacking 元學習器
- **WBC 感知**: 用球數限制、陣容 RSI、貝葉斯收縮已實作
- **市場整合**: CLV 優化、蒸氣偵測、執行滑點建模已實作
- **機構安全閘**: 7 階段決策閘防止假邊際效益下注

### 2.2 架構弱點 ❌

#### 方法論層面
| 弱點 | 說明 | 優先級 |
|---|---|---|
| 回測樣本嚴重不足 | 2023 WBC 僅 ~20 場，統計功效不足 | P0 |
| 合成數據混入回測 | 污染歷史驗證的有效性 | P0 |
| 固定集成權重 | 違背自適應集成原則 | P1 |
| 獨立二元 Poisson | 忽略兩隊得分的同場相關性 | P2 |
| 貝葉斯先驗過於簡單 | 僅用聯盟平均，未利用歷史對戰先驗 | P2 |

#### 技術實作層面
| 弱點 | 說明 | 優先級 |
|---|---|---|
| 無實時資料管道 | 依賴靜態 CSV，無法獲取賽前最後 48h 資訊 | P1 |
| 牛棚疲勞模型線性化 | `min(1, pitch_3d/100)` 忽略非線性疲勞與連鎖效應 | P1 |
| BankrollState 無持久化 | 系統重啟後資金狀態歸零 | P1 |
| API 無離線備援 | 外部 API 中斷時無降級策略 | P2 |

#### 系統架構層面
| 弱點 | 說明 | 優先級 |
|---|---|---|
| WBC/MLB 無架構分離 | 兩種場景共用同一套配置，難以獨立優化 | P1 |
| 決策閘串行執行 | Phase 1b-1e 可並行，延遲 200ms → 80ms | P2 |
| 缺乏 A/B 測試基礎設施 | 無法安全測試新模型版本 | P2 |
| 缺乏模型漂移監控儀表板 | 無可視化的效能衰退偵測 | P2 |

## 三、WBC vs MLB 建模差異

| 維度 | WBC | MLB | 建模影響 |
|---|---|---|---|
| 樣本量 | 5-7 場 | 162 場 | WBC 需更強的先驗收縮 |
| 用球數限制 | 65/80/95 強制上限 | 無強制限制 | WBC 牛棚主導 60%+ 比賽 |
| 陣容穩定性 | 國家隊短期磨合 | 球季累積化學反應 | WBC 需磨合度折扣係數 |
| 主客場 | 中立球場（優勢=0） | 主場 +0.035 勝率 | WBC Elo 需歸零主場因子 |
| 市場深度 | 流動性低 | 流動性高 | WBC 市場效率較低，邊際效益空間更大 |
| 歷史數據 | 3-4 屆（稀少） | 100+ 年 | WBC 需遷移學習策略 |

**建議**: 實作 `WBCAdapter` / `MLBAdapter` 適配器模式，從共享 `BaseModel` 派生。

## 六、回測框架改進建議

### 目標回測數據集
```
Layer 1: WBC 歷史 (~78 場)
  - 2009 WBC (26 場)
  - 2013 WBC (16 場)
  - 2017 WBC (16 場)
  - 2023 WBC (20 場)

Layer 2: 類 WBC 代理數據
  - 2020 東奧棒球 / 2008 北京奧運
  - Premier12 錦標賽
  - NPB/KBO 淘汰賽

Layer 3: MLB 短系列賽（用球數限制相似）
  - 季後賽分區系列賽 / 野外卡賽
```

### 驗證成功標準
```
最低標準（可信回測）:
  ✓ 樣本數 ≥ 50 場 WBC
  ✓ Brier Score < 0.23
  ✓ ROI > 0%（打平 vig）
  ✓ Sharpe Ratio > 0.5

理想標準:
  ✓ 樣本數 ≥ 200 場
  ✓ Brier Score < 0.21
  ✓ ROI > 3%
  ✓ CLV Rate > 60%
  ✓ 跨屆 WBC 每屆 ROI 均為正
```
